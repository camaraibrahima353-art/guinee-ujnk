Configurer Supabase

Base de données temps réel pour votre pharmacie

1

Allez sur supabase.com → connectez-vous → New Project

2

Dans SQL Editor → New Query → copiez-collez le SQL du fichier config.js (entre /* et */) → cliquez Run ▶

3

Dans Project Settings → API → copiez Project URL et la Publishable key
